PWA DEEP LINK SUPPORT
---------------------
Added URL parameter handling for PWA navigation:

Deep Link Parameters:
- ?tab=home|share|settings  - Jump to specific tab
- ?game=wordle              - Launch a game directly
- ?play=connections         - Alias for ?game
- ?action=log               - Open log sheet (existing)
- ?share-target=true        - Receive shared text (existing)

PWA Shortcuts (long-press app icon):
- Log Game       - Open log sheet
- Share Results  - Go to share tab
- Play Wordle    - Launch Wordle directly

- isStandalone() returns true when running as installed PWA
