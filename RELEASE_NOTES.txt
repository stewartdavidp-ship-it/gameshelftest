Game Shelf v1.2.0.12 Release Notes
===================================
Released: January 22, 2026

CHANGES FROM v1.2.0.11:
-----------------------
- Added 30-second minimum time check for tap-to-log prompt
  - If user was away < 30 seconds, no prompt (just browsed menu)
  
- Added "already processed" clipboard tracking
  - After logging a game, remember that clipboard content
  - If same content is still there on next return, skip prompt
  - Prevents false prompts when visiting another game after logging
