Game Shelf v1.2.0.22
====================
Release Date: January 22, 2025

UAT MODE & FEEDBACK SYSTEM
--------------------------
This release adds a comprehensive User Acceptance Testing (UAT) mode 
designed for collecting feedback during beta testing.

New Features:

1. UAT Mode Toggle (Settings > Developer)
   - Enables floating feedback button
   - Shows "UAT Mode" badge in top-left corner
   - Captures debug log of user actions

2. Floating Feedback Button
   - Always-visible purple button (bottom-left)
   - One-tap access to report issues/enhancements
   - Only visible when UAT Mode is enabled

3. Auto-Captured Debug Context
   - Current tab and app version
   - Wallet balance, game count, streak
   - Device/platform info
   - Last 5 activity log entries
   - Console errors captured during session
   - Recent user actions (tab switches, game logs, launches)

4. Enhanced Issue Reports
   - Debug context automatically attached when UAT mode is on
   - "Include debug log" checkbox (default: ON)
   - Source marked as "pwa-uat" for UAT submissions
   - App version now pulled from meta tag (not hardcoded)

Technical Details:
- UAT debug log stores up to 50 recent actions
- Console.error intercepted and logged
- Context captured on feedback form open
- Firebase structure: reported-issues/{id}/debugContext
