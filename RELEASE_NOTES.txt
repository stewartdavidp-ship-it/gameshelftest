Game Shelf v1.2.0.9 Release Notes
==================================
Released: January 22, 2026

CHANGES FROM v1.2.0.8:
----------------------
- Fixed clipboard prompt showing for GS Originals (Quotle/Slate/Rungs/Word Boxing)
  - showTapToLogCardForClipboardCheck() now skips GS Originals
  - checkClipboardOnReturn() now skips GS Originals
  - tapToLogClipboardWithDiff() now skips GS Originals
  
- Fixed false prompts when clipboard couldn't be captured on leave
  - CLIPBOARD_UNKNOWN state now requires matching lastLaunchedGame
  - More conservative: only prompt if result matches the game we launched
  
- Added detailed console logging for debugging clipboard flow:
  - Logs clipboard state on leave and return
  - Logs decision points for troubleshooting

PREVIOUS (v1.2.0.8):
--------------------
- Redesigned clipboard detection with "diff" approach
- Captures clipboard state when leaving the app
- Compares on return to detect new content
