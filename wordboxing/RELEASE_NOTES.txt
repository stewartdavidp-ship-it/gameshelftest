Word Boxing v1.0.15
==================
Release Date: February 3, 2026

DEBUG FEATURE - "Stuck?" Button
-------------------------------
- Added floating "Stuck?" button in bottom-left corner during active games
- Button only appears during game phases (not in lobby or match results)
- Tapping opens confirmation modal explaining what will happen
- On confirm:
  - Captures sanitized game state snapshot (no actual words/letters)
  - Submits debug report to Firebase at `debug-reports/`
  - Cleans up local game state and listeners
  - Returns user to lobby with confirmation message

Debug report includes:
- Game code, phase, round, turn state
- Connection status and player roles  
- Progress indicators (counts only, not actual content)
- Device info (user agent, screen size)
- Timestamp and app version

This helps diagnose multiplayer sync issues that are hard to reproduce.
